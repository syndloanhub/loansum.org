Introducing loansum

Loansum is the initial open source offering from SyndLoanHub, LLC. Loansum is the first open source library designed from scratch to accurately model syndicated bank loan cash flows and valuation. Financial software developers can leverage loansum software to construct or extend applications with bona fide bank loan support.

 Loansum is open source

Native bank loan software is notoriously complex and expensive. Only a handful of software vendors model loans accurately with sufficient support to produce detailed cash flows with accompanying explanations. Loansum was developed to establish a free and open source loan software library around which a community of stakeholders would evolve to continuously extend and improve the software over time. Loansum is released as Open Source Software under the Apache v2.0 license. This is a widely used, business-friendly, license.
