Running the calculator

The best way to become acquainted with loansum functionality is to play with the calculator application by clicking on Calculator. The calculator contains a variety of example loans that may be selected in order to examine loan data and generate cash flows with accompanying explains. Loans may be edited to observe the effect of changes and new loans and trades may also be created. A set of short videos which demonstrate the basic usage of the calculator application may be accessed by clicking on Demos.

 Obtaining loansum

Loansum is available in Maven Central: 

<dependency>
    <groupId>com.syndloanhhub.loansum</groupId>
    <artifactId>loansum-pricer</artifactId>
    <version>0.1.0-beta</version>
</dependency>

This dependency on the loansum-pricer module will provide access to all of loansum's cash flow functionality. Loansum is also released to GitHub through the standard repository release pages. Source code can be obtained directly from GitHub. To do this you will need Git. Simply clone the repository locally: 

git clone https://github.com/syndloanhub/loansum.git

Finally, since loansum extends Strata, the OpenGamma Strata site contains a great deal of relevent and useful documentation as well which we do not attempt to replicate here.

 Code examples

The best source of example code is the loansum-pricer module unit test code. This code is located in the source file ProratedLoanTradePricerTest.java. The unit test code demonstrates how to construct loan and associated trade objects, generate prorated versions of these objects, and use pricer instances to calculate cash flows, accompanying explains, and present value.
