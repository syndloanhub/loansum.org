Running the calculator

Loansum is the initial open source offering from SyndLoanHub, LLC. Loansum is the first open source library designed from scratch to accurately model syndicated bank loan cash flows and valuation. Financial software developers can leverage loansum software to construct or extend applications with bona fide bank loan support.

 Obtaining loansum

Native bank loan software is notoriously complex and expensive. Only a handful of software vendors model this product accurately with sufficient support to produce detailed cash flows with accompanying explanations. Loansum was developed to establish a free and open bank loan software library around which a community of stakeholders could extend and improve over time. Loansum is released as Open Source Software under the Apache v2.0 license. This is a widely used, business-friendly, license.
