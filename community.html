Current Status

Loansum is currently at release version 0.1. Pending initial feedback on the library and functionality, version 1.0 will be released. The direction of the library will depend on community priorities and more general SyndLoanHub product requirements. It is likely that a near term priority will be comprehensive import and export of Syndicated Loan FpML. Please email any feedback or questions to info@syndloanhub.com.

 Support

At this time, general loansum support is provided via email only. Please send any questions or issues to support@syndloanhub.com. In the near future, the loansum.org web site will be extended with a community forum to allow the exchange of information between users and contributors.

 Contributing to loansum

Contributions to loansum are welcome and encouraged. Bug reports should be raised as GitHub Issues. Please provide as much detail as possible, in particular sample code that reproduces the reported bug. Feature requests are also welcome. Please email feature requests, comments and questions to info@syndloanhub.com. If you have a patch to fix a bug or add a feature, please follow standard GitHub protocol:

  1. Fork loansum
  2. Create a pull request against the master repository
  3. If all is well, your patch will be accepted and merged
