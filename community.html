Current Status

Loansum is currently at release version 0.1.1-beta. Subsequent minor beta releases and patches will be made as issues are reported. Version 1.0.0 (non-beta) will be released once initial feedback on the library and functionality has been considered and relevent changes have been implemented. The direction of the library will depend on community priorities and more general SyndLoanHub product requirements. It is likely that a near term priority will be comprehensive import and export of Syndicated Loan FpML. Please communicate any feedback or questions to the SyndLoanHub email group.

 Support

General loansum support is provided via the SyndLoanHub email group. The SyndLoanHub email group serves as a general community forum to allow the free exchange of information between users and contributors. Specific reproduceable bugs may be raised as GitHub Issues.

 Contributing to loansum

Contributions to loansum are welcome and encouraged. Bug reports should be raised as GitHub Issues. Please provide as much detail as possible, in particular sample code that reproduces the reported bug. Feature requests are also welcome. Please communicate feature requests, comments and questions to the SyndLoanHub email group. If you have a patch to fix a bug or add a feature, please follow standard GitHub protocol:

  1. Fork loansum
  2. Create a pull request against the master repository
  3. If all is well, your patch will be accepted and merged
